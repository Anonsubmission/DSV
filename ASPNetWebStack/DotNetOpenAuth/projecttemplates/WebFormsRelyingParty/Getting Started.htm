Your OpenID and InfoCard relying party web site is nearly ready to go. You just need to create your SQL database where user accounts will be stored. Just build and start your web site and visit Setup.aspx. You'll get further instructions there.

Creating your database is almost totally automated, so it should be a piece of cake. Note however that creating the database requires Modify permissions in your web site's App_Data directory for the security account that creates databases on your computer. If you get a permissions or file write error from setup.aspx, try temporarily adding Everyone: Modify permissions to your App_Data folder just long enough to create your database.
